# Bitburner-scripts
Here is the script I used in [Bitburner](https://bitburner-official.github.io/).

## Contents
| ファイル名 | 役割 | 要件 |
| :--- | :--- | :--- |
| `quick-script.js` | 基礎攻略で使用するスクリプトを作成するもの。 | 1.60GB |
| `Overview/hack-XPmonitor.js` | 1秒毎のHack経験値獲得量をOverviewに表示します。 | 6.10GB |
| `Search-server/deep-scan.js` | 全ての階層のサーバーをスキャンします。 | 3.80GB |
| `Search-server/find.js` | 指定したサーバーに移動するためのコマンドを出力します。 | 1.80GB |
| `Terminal/check-script.js` | 指定したスクリプトの稼働数を調べます。 | 2.00GB |
| `Terminal/hack.js` | 指定したサーバーをハックします。 | 2.30GB |
| `Terminal/launcher.js` | 指定したスクリプトを限界のスレッド数で稼働させます。 | 3.15GB |
| `Terminal/xp-farm.js` | 特定のサーバーを成長させ経験値を稼ぎます。 | 2.10GB |
