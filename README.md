# Bitburner-scripts
Here is the script I used in [Bitburner](https://bitburner-official.github.io/).

| ファイル名 | 役割 | 必要RAM |
| :--- | :--- | :--- |
| `hack-XPmonitor.js` | 1秒毎のHack経験値獲得量をOverviewに表示します。 | 6.10GB |
| `deep-scan.js` | 全ての階層のサーバーをスキャンします。 | 3.80GB |
| `find.js` | 指定したサーバーに移動するためのコマンドを出力します。 | 1.80GB |
| `check-script.js` | 指定したスクリプトの稼働数を調べます。 | 2.00GB |
| `hack.js` | 指定したサーバーをハックします。 | 2.30GB |
| `launcher.js` | 指定したスクリプトを限界のスレッド数で稼働させます。 | 3.15GB |
| `xp-farm.js` | 特定のサーバーを成長させ経験値を稼ぎます。 | 2.10GB |
